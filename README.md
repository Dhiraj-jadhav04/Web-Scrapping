# Quotes Web Scraper (Python)

This project scrapes quotes from https://quotes.toscrape.com using Python.

## Features
- Scrapes multiple pages automatically
- Saves HTML files locally
- Extracts quote text, author, and tags
- Filters quotes related to "life"

## Technologies Used
- Python
- requests
- BeautifulSoup (bs4)
- lxml

## How to Run
1. Install dependencies:
   pip install requests beautifulsoup4 lxml
2. Run the script:
   python scraper.py

## Learning Outcome
- Hands-on experience with web scraping
- HTML parsing using CSS selectors
- Data extraction and filtering
